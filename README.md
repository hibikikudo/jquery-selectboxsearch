## Usage

1. Include jQuery:

	```html
	<script src="http://ajax.googleapis.com/ajax/libs/jquery/1.8.3/jquery.min.js"></script>
	```

2. Include plugin's code:

	```html
	<script type="text/javascript" src="jquery.selectboxsearch.js"></script>
	```

3. Call the plugin:

	```javascript
        $('#element').selectboxsearch('#target');
	```
